File Structure
viewer/
  __init__.py
  main.py         # Entry point, game loop
  map_loader.py   # Load layered map data
  renderer.py     # Tile/layer/parallax drawing
  camera.py       # Pan, zoom, parallax offset
  fog_of_war.py   # Visibility + LOS raycasting
  game_engine.py  # State machine: explore/combat
  combat.py       # D20 rules, turn order, attacks
  entities.py     # Player, Enemy, NPC classes
  ai.py           # Enemy behavior (chase/patrol)
  ui_overlay.py   # HUD, HP bars, combat log, minimap
  config.py       # Settings, keybinds, constants
Rendering Pipeline (per frame)
1 Camera calculates visible tile range + parallax offset
2 Draw z-levels below current (dimmed 50%)
3 Draw current z-level terrain + structures
4 Draw entities (tokens, furniture, doors)
5 Apply fog of war mask (black unexplored, dim explored, clear visible)
6 Draw UI overlay (HP bars, combat log, minimap, stats)
7 Flip display buffer (vsync)
GPU Utilization (RTX 5090)

OpenGL Accelerated

  • Tile blitting via GPU surfaces
  • Layer compositing on GPU
  • Fog of war as alpha overlay
  • Smooth zoom/pan transforms
  • VSync at native refresh rate

Future CUDA potential

  • GPU-accelerated LOS raycasting
  • Real-time dynamic lighting
  • Particle effects (fire, smoke)
  • Large map pathfinding (A* on GPU)
