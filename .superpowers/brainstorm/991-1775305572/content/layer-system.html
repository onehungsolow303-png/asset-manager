How z-levels are stored and used across the system

Data Structure (per z-level):
ZLevel:
  z: int                    # 0 = ground, +N = above, -N = below
  terrain_color: uint8[H,W,3]  # RGB per pixel
  walkability: bool[H,W]    # passable tiles
  structure_mask: bool[H,W] # walls/floors
  entities: list[Entity]    # furniture, doors, stairs, NPCs, enemies
  transitions: list[Transition]  # stairs/ladders linking to other z-levels
Layer Visibility Rule:
Player at z=0 (ground floor):
  Show: z=0 (current), z=-1 (below, dimmed)
  Hide: z=1+ (roof, upper floors)

Player at z=-1 (basement):
  Show: z=-1 (current), z=-2 (below, dimmed)
  Hide: z=0+ (everything above)

Player at z=2 (upper floor):
  Show: z=2 (current), z=1 (below, dimmed)
  Hide: z=3+ (above), z=0 and below (ground)
