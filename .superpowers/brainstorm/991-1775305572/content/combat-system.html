Section 2: Tabletop RPG Combat System

D20-based rules (5e-inspired, no trademarked names)

Token Types
P
Player Token
Blue circle. WASD movement. You control this.
E
Enemy Token
Red circle. Auto-placed by generator. Basic AI.
N
NPC Token
Gold circle. Non-hostile. Interact with E key.
Creature Stats (all tokens)
HP: Hit Points (health pool)
AC: Armor Class (hit threshold)
STR: Melee attack + damage mod
DEX: Initiative + ranged + AC mod
CON: HP bonus
SPD: Movement tiles per turn (default 6)
ATK: Attack dice (e.g. 1d8+3)
INT/WIS/CHA tracked but not used in v1
Combat Flow
1. Exploration Mode
→
2. Enemy in range → Initiative roll
→
3. Turn-based combat
→
4. All enemies dead → back to exploration

Exploration Mode

  • Free WASD movement (real-time)
  • Fog of war reveals as you explore
  • Line-of-sight blocks through walls
  • E key to interact (doors, NPCs, stairs)
  • Camera follows player, parallax layers shift

Combat Mode (turn-based)

  • Grid snaps to tiles (5ft squares)
  • Click tile to move (within SPD range)
  • Click enemy to attack (within weapon range)
  • d20 + STR/DEX mod vs target AC
  • Hit: roll damage dice, subtract from HP
  • Enemy AI: move toward player, attack if adjacent
  • HP bar shown above each token
Keyboard Controls
Movement
WASD / Arrow keys — Move player
Mouse wheel — Zoom in/out
Middle mouse drag — Pan camera
Interaction
E — Interact (door/NPC/stairs)
Space — End turn (combat)
Tab — Toggle perspective mode
View
[ / ] — Force z-level up/down
F — Toggle fog of war
G — Toggle grid overlay
Esc — Menu / quit
